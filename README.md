# 元素种类
块元素(block element)和内联元素(inline element)  
块元素独立成块，与其他内容自带分界
内联元素嵌入在其他内容中

# 类
        <h1 class= "class1 class2 class3">
可以为元素指定类，为一类元素，或为元素中的一类统一设置属性  
越靠后列出的规则优先级越高（在类优先级相同的情况下）
文字属性名称|功能|备注
---|---|---
font-family|指定字体|sans : 表否定  serif : 衬线  monospace : 等宽 
font-size|字体大小|单位px
color|文本颜色|
font-weight|字体粗细|
text-decoration|文本风格|上下划线、删除线
